## これは何？

WebGPU(JSライブラリ)をKotlin/JSから直接叩いて描画してみようというやつ

今のところdogeブロックBBが描画される

Kotlin流に書けるように工夫してるのでmainとか見てみると面白いかもしれない

## 実行方法

JRE(JDK) 17+を用意してください
```bash
git clone https://github.com/16bittoSeisuu/webgputest.git
cd webgputest
./gradlew jsBrowserDevelopmentRun
```
Windowsなら
```powershell
gradlew jsBrowserDevelopmentRun
```
